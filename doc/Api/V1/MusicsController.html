    <EU012> Eu como usuário quero ser capaz de ver a página da música para adicionar aos favoritos, avaliar e ver outros usuários que favoritaram a música. Mostra ao usuário a página de uma determinada música

def show
  music = Music.find(params[:id])

  if music.ratings.blank?
    average_rating = 0
  else
    average_rating = music.ratings.average(:value).round(1)
  end

  music.average = average_rating

  render json: music, status: :ok
rescue StandardError => e
  render json: { message: e.message }, status: 404
end
